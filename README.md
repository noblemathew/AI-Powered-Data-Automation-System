# AI-Powered-Data-Automation-System

AI-Powered-Data-Automation-System is a Python-based voice-enabled AI assistant inspired by J.A.R.V.I.S. from Iron Man. The system combines speech recognition, AI-powered conversations, web automation, and system control into a futuristic interactive assistant.

## Features

- 🎙️ Voice Recognition and Speech Response
- 🤖 Gemini 1.5 Powered AI Conversations
- 🌐 Web Search Automation
- 📰 Live News Reading
- ⏰ Date and Time Queries
- 📚 Wikipedia Search
- 💻 System Diagnostics
- 📸 Screenshot Capture
- 🔊 Volume Controls
- 🚀 Application Launcher
- 🔒 Lock, Restart, and Shutdown Controls
- 🎨 Futuristic J.A.R.V.I.S.-Inspired UI

## Technologies Used

- Python
- HTML/CSS/JavaScript
- SpeechRecognition
- pyttsx3
- Gemini 1.5 API
- BeautifulSoup
- Requests
- PyAutoGUI
- Wikipedia API
- psutil
